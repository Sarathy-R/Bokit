# Bokit

  Bokit is an online based movie ticket booking application.
Made with pure HTML,CSS and JS. No use of  any JS library/plugins.


![alt-text](https://raw.githubusercontent.com/sarathy711/Bokit/master/bokit.PNG)

## Installation

No installation required

```
Clone or Download the code
```
## Usage


1.) Open the html file in the browser.

2.) Select the movie and available seats.

3.) Check out your ticket cost.


Note:-
       The webpage has inbuilt session storage feature.
